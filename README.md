# how-I-met-the-internet
Prototype for a performace from the artist Olga Karaseva with websockets and Touchdesigner


## Installation:

- Install Node.js and Git
- git clone or download the zip file
- Open the terminal and locate on the folder
- type npm install

### To run the server:

- Open the terminal and locate on the folder
- type node server.js

### The page should be working on: 

http://localhost:7071/     (type it on the browser)

http://yourWifiIpAddress:7071/    for remote clients

### Based on https://github.com/Koproduktionslabor/websocketsTouchdesigner
